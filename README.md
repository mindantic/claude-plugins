# Mindantic Claude Code Plugins

Official plugin marketplace for [Mindantic](https://github.com/mindantic).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [postical](https://github.com/mindantic/postical) | Content engine for creating, optimizing, and distributing blog content at scale |

## Setup

Add this marketplace to Claude Code:

```bash
claude plugins marketplace add mindantic/claude-plugins
```

## Install Plugins

```bash
claude plugins install postical@mindantic
```
